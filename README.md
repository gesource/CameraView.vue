# CameraVue.vue

Webブラウザにカメラの映像を表示します。

## 使用ライブラリ

* [Vue.js](https://github.com/vuejs/vue)
* [webrtc-adapter](https://github.com/webrtc/adapter)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
